Setup

  1. First of all, sign up for GitHub and verify your email address.
  2. Next, you need to create a repository for your files to go in. In the Repository name box, enter username.github.io, where username is your username. So for example, decisacter.github.io. Also check Initialize this repository with a README, Add license and then click Create repository.
  3. After that, commit your file using terminal. Make sure you have an index.html file in the root. This is your homepage. It may take a few minutes for your website to go live.
    4. 
            $user = "decisact"
            git config --global user.email "$user-@example.com"
            git config --global user.name $user
            git init
            git remote add origin "https://github.com/$($user)/$($user).github.io.git"
            git add .
            git commit -m "first commit"
            git push -u origin master
  4. Now navigate your browser to username.github.io to see your website online. For example, for the username decisact, go to decisact.github.io
